## ▶️ How to Run

### Compile and run any file:

```bash
# Navigate to the array folder
cd array

# Compile
javac Q01_SumOfElements.java

# Run
java Q01_SumOfElements
```

### Example Output (Q01):
```
Array:
10 20 30 40 50
Sum of all elements: 150
```

## 📌 Notes

- Every file is **self-contained** — no external libraries needed (except standard Java).  
- Each file includes a **sample array** so you can run it immediately without any user input.  
- Comments use `//` to describe the question and logic inline.  
- Questions go from **easy → medium → hard** roughly in order.
